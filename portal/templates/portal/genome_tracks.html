    JASPAR UCSC tracks

    UCSC Genome Browser track data hub containing genome-wide binding site predictions for TF profiles in the JASPAR CORE vertebrates collection.

    How-to

    1. Direct link

    View the hg19 and hg38 tracks directly in the UCSC Genome Browser.

    2. Track data hubs

    Insert the JASPAR track data hub URL into your own UCSC Genome Browser session via "MyData => Track Hubs" by pasting the hub URL in the "URL:" box and clicking on "Add Hub".

    JASPAR track data hub URL: http://expdata.cmmt.ubc.ca/JASPAR/UCSC_tracks/hub.txt

    3. Download

    Underlying TFBS prediction data is available for download and further analysis: http://expdata.cmmt.ubc.ca/JASPAR/downloads/UCSC_tracks/2018/
