{JASPAR RESTful API}

This API provides easy-to-use REST web interface to query/retrieve matrix profile data from the latest version of JASPAR database. The API comes with a human browsable interface and also programmatic interface, which return the results in six different formats, including json, jsonp, jaspar, meme, transfac, pfm, yaml and html.
